# AI Virtual Teaching Assistant

A standalone web application for managing virtual teaching sessions with AI-powered note generation.

## Project Structure

- `frontend/` - Standalone HTML/CSS/JS frontend
- `backend/` - Node.js/Express API server

## Setup

### Backend
```bash
cd backend
npm install
# Configure .env with MongoDB connection
npm run dev
```

### Frontend
Simply open `frontend/index.html` in your browser.

## Features
- Meeting management
- AI-powered note generation
- Schedule tracking
- Real-time updates

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, MongoDB
- AI: OpenAI API
